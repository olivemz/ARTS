## tips
### shell script double square bracket
http://mywiki.wooledge.org/BashFAQ/031 \
if [[  -f $file  ]]; then echo "$file is a regular file"
same as
if [ -f "$file"]; then echo "$file is a regular file"
### shell script source things
https://ss64.com/bash/source.html
```
  source sample.sh
```
is same as
```
  . sample.sh
```
